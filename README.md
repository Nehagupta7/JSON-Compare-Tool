# JSON Compare Tool

A modern React-based tool for comparing JSON objects with deep nested diff detection, side-by-side visualization, formatted views, and automatic change highlighting.

## Features

* Compare two JSON objects instantly
* Deep comparison of nested objects and arrays
* Side-by-side diff view
* Human-friendly formatted view
* Automatic change counting
* Expand/Collapse changed sections
* Dark mode support
* JSON validation with error handling
* Responsive and developer-friendly UI

## Tech Stack

* React
* JavaScript
* Tailwind CSS v4

## Use Cases

* API response comparison
* Audit log review
* Configuration validation
* Data migration verification
* Debugging object changes
